vim-config
==========

My personalized vim configuration.

Setup Instructions:
1. Clone this repository into the ~/.vim directory.

2. Symlink ~/.vimrc so that vim loads with the custom .vimrc inside
   this folder:
	ln -s .vim/.vimrc .vimrc

2. Create dirs for backup and tmp files
	a. Create ~/.vim/backup
	b. Create ~/.vim/tmp
